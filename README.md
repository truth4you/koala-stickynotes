# Sticky Notes
This is test project to write memos using HTML5 Canvas and React.

## Installation
Install npm packages

        npm install

## Start for Development
        npm start
        node src/server.js
## Start for Release
        npm run-script build
        node src/server.js
        
## Description
### Used Stacks
* Frontend: React, Konva
* Backend: Node, Express, Socket.io
